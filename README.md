Bethara Engineering Service is a comprehensive automotive service center located in Pannipitiya, Sri Lanka. Our modern web application showcases our core services and commitment to quality, built to deliver a seamless user experience across all devices.

- 🚿 **Car Wash** - Professional exterior & interior cleaning
- ⚙️ **Auto Parts** - Genuine and OEM parts supply
- 🔧 **Auto Service** - Comprehensive vehicle maintenance
- 🛠️ **Engineering** - Precision fabrication and machining
- 🌗 **Custom Theming** - Built-in Light and Dark modes
- 📱 **Mobile Optimized** - Perfect experience on any screen

---

### 🖥️ Website Features
- **Responsive Design** - Optimized for all devices (desktop, tablet, mobile)
- **Interactive Gallery** - Lightbox viewer for workshop photography
- **Custom Design System** - Brand-aligned color palette (Graphite, Lime Moss, Lemon Lime)
- **Contact Integration** - Google Maps, quick-dial links, and WhatsApp
- **Modern Animations** - Smooth transitions and scroll effects

### 🚀 Quick Start (Local Development)

1. **Clone the repository**
   ```bash
   git clone https://github.com/dinethsadee01/Auto-Service-Hub.git
   cd Auto-Service-Hub
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start local development server**
   ```bash
   npm run dev
   ```

4. **Production Build**
   ```bash
   npm run build
   ```
